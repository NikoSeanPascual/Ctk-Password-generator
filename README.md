# Password-Gen-GUI

A modern and user-friendly **Password Generator GUI** built with **CustomTkinter**. This application allows you to create strong, customizable passwords quickly, with options for length, uppercase, lowercase, numbers, and symbols.

---

## Features

- **Adjustable password length:** Choose between 8 and 32 characters using a slider.  
- **Customizable character options:** Include or exclude uppercase letters, lowercase letters, numbers, and symbols.  
- **Generate passwords instantly:** Click a button to generate a password with the selected options.  
- **Copy to clipboard:** Easily copy the generated password with one click.  
- **Dark mode interface:** Sleek, modern GUI for a better user experience.

---

## How It Works

1. **Password Length:**  
   Use the slider to set how long you want your password to be. The default is 12 characters.  

2. **Character Options:**  
   - Uppercase Letters (A-Z)  
   - Lowercase Letters (a-z)  
   - Numbers (0-9)  
   - Symbols (!@#$ etc.)  

   You **must select at least one option** to generate a password.

3. **Generate Password:**  
   The app combines the selected characters and randomly picks characters to create a secure password of the chosen length.  

4. **Copy Password:**  
   Press the **COPY** button to copy the password to your clipboard for easy use.
